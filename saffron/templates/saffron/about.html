About SAFFRON

SAFFRON — Signal-peptide Assessment Framework For Ranking Of N-terminal edits — is a sibling tool to AMBER.

It uses SignalP 6.0 (fast mode) to predict signal peptides, designs ABE/CBE guides that introduce missense changes in the predicted SP region, and reports Δ SignalP scores for each mutant.

SignalP is run as a local CLI (Python 3.10 sidecar). When the binary is not installed, SAFFRON falls back to a deterministic mock predictor so the UI can be developed and tested.

Guide outcomes are annotated with CLIN_SIG (Ensembl VEP) and potentially pathogenic flags from Gutierrez Guarnizo et al., NAR Genom Bioinform (2023) when the UniProt accession and amino-acid change appear in their published catalogue. Matching “Yes” rows are highlighted in light red in the results table.
